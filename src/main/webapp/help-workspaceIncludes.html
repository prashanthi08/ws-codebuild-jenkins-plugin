Ant fileset patterns to specify files to be zipped. Multiple patterns can be specified separating with commas(,). Leaving blank results all files will be zipped.

Some examples:

  • **/*.java: All files with '.java' extension. This includes files in subdirectories.
  • *.sh: Files with '.sh' extension in the root directory. This doesn't include files in subdirectories.
  • subdir/: All files in the 'subdir' directory.
