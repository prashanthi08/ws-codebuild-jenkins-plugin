Optional build cache modes override.
  • If cache mode is set to LOCAL_SOURCE_CACHE, this is mode caches Git metadata for primary and secondary sources.
  • If cache mode is set to LOCAL_DOCKER_LAYER_CACHE, this is mode caches existing Docker layers.
  • If cache mode is set to LOCAL_CUSTOM_CACHE, this is mode caches directories you specify in the buildspec file.
Specified cache modes must be enclosed in brackets. Example: "[LOCAL_SOURCE_CACHE, LOCAL_DOCKER_LAYER_CACHE]"
